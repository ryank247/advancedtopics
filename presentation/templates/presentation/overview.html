Introduction

What is CS50 web development?

In "HarvardX: CS50's Web Programming with Python and JavaScript" we learn about various topics on how to build a website using various coding methods and topics we will talk about today.

This course is free through EdX and is taught by Harvard. Brian Yu (my glorious king, pictured on the right) is the one who specifically teaches us about web development giving us hour long lectures about certain topics with hands-on applications. In addition to this, for every unit we have projects based upon a certain topic, such as python. We will go over this later.

Some of the topics include:
- HTML, CSS***
- Git***
- Python***
- Django***
- SQL, Models, and Migrations***
- JavaScript
- User Interfaces
- Testing, CI/CD
- Scalability and Security

The ones that have stars are the ones we have got through in the first semester and will be talking about.
Also in this course, upon completion and purchase, there is a certificate that you can obtain from HarvardX, stating completion.
