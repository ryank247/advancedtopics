By using this link, I have allowed the user to go to a completely new page, which you can see we are doing in this presentation.

Heres how we created it:
  • First, set up an html page for your link in the templates category, then also use a template like this in a separate html page to go to the new one "a class="nav-link" href=" url 'newpage.html' "More Cool Stuff Link a" (I left out key symbols to not intefere with code)
  • Next you would want to create a function to do so in views.py which would look like this:
    def newpage(request):
    return render(request, "presentation/newpage.html")
  • Finally, you would want to set up a url to complete this request, to do so, in urls.py:
    path("newpage", views.newpage, name="newpage")

  • One of these include {load static}, which loads the static parts of your code, that being the stylesheet for CSS
  • Another could include {for blank in blank} and {endfor} which is a for loop
  • Finally, one last one of these lines that Django includes is {block body} and {endblock} which helps with the design of the webpage by creating certain blocks.
